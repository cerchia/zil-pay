### Установка

```bash
$ npm install # установить зависимости
```

### Сборка

* For develop
```bash
$ npm run serve # Запустить разширение для разработки.
```

* Для production:
```bash
$ npm run build # Создать production билд.
```

### Запуск тестов
```bash
# Запустить все тесты.
$ npm test:unit

# Запустить тесты для разработки.
$ npm run test:watch

# Проверить код на lint.
$ npm run lint
```
