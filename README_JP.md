### 建物

* 開発のため
```bash
# 依存関係をインストールする
$ npm install

# すべてのファイル拡張子を提供します。
$ npm run serve
```

* 本番用：
```bash
# すべてのアプリと拡張機能ファイルを作成し、最適化します。
$ npm run build
```

### テストを実行する
```bash
# 単体テストを実行する
$ npm test:unit

# 開発者向けのテストを見る。
$ npm run test:watch

# lint ts、vueファイルを確認してください。
$ npm run lint
```
