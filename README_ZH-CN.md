### 正在安装

为发展建设和服务。
``` bash
# 安装依赖
$ npm install

# 服务所有文件扩展名。
$ npm run serve

---

测试项目。
``` bash
# 运行单元测试
$ npm test:unit

# 观看开发测试。
$ npm run test:watch

# 检查lint js，vue文件。
$ npm run lint

```

---

敌人生产建设和部署。
``` bash
# 构建所有应用程序和扩展文件并对其进行优化。
$ npm run build
```
