### Instalando

### edificio

* Para desarrollar
```bash
# Instalu dependojn
$ npm install

Serve all files extension.
$ npm run serve
```

* Para la producción:
```bash
# Konstruu ĉiujn dosierojn kaj etendajn dosierojn kaj optimumigante ĝin.
$ npm run build
```

### Ejecutando las pruebas
```bash
# aranĝu ekzemplerojn
$ npm test:unit

# Rigardi provadon por dev.
$ npm run test:watch

# Kontrolu lint ts, vue dosierojn.
$ npm run lint
```
